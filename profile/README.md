## 🎓 Thesis Project

### [`plataforma-desafios-ml-fac`](https://github.com/juanguillermo-dev/plataforma-desafios-ml-fac)
**ML Challenges Platform — Colombian Air Force × Universidad de los Andes**

A production-grade web platform for managing machine learning competitions focused on aerial image analysis. Organizers publish challenges with datasets; participants submit trained models that are automatically evaluated asynchronously and ranked on a leaderboard.

**Stack:** FastAPI · PostgreSQL · MinIO · Celery + Redis · Next.js · Docker · TensorFlow · PyTorch · JWT Auth
